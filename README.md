### Features

- aiohttp
- auto filter windows forbidden file names (CON, NUL, COM1...  etc.)
- non blocking upload and download files
- download page with file info

### Requirements

- Python 3.6+
- AioHTTP 3.6+
- aiohttp-jinja 1.2+
- Jinja2 2.10+
